# 🎨 Projeto: Loja de Materiais Pedagógicos

Este projeto é o Front-end de uma plataforma de e-commerce focada em recursos didáticos lúdicos. O objetivo atual é a entrega da estrutura semântica padronizada (HTML5) antes da aplicação de estilos (CSS).

## 🛠️ Tecnologias e Conceitos Aplicados

Nesta primeira entrega, o foco total foi em **HTML5 Semântico**, Acessibilidade e SEO:

- **Semântica:** Uso de tags estruturais como `<header>`, `<main>`, `<section>` e `<footer>` para uma hierarquia clara.
- Uso de `aria-label` e `aria-hidden` para garantir que leitores de tela entendam ícones visuais.
- **SEO:** Implementação de `meta description` estratégica e títulos (`h1`) otimizados para busca.
- **Formulários:** Criação de páginas de Login e Cadastro com validação nativa (`required`, `type="email"`) e agrupamento lógico com `<fieldset>`.

## 📁 Estrutura de Pastas

O projeto segue um padrão de organização profissional:
- `/assets/images`: Imagens dos produtos e logo.
- `/assets/icon`: Favicon do site.
- `index.html`: Página principal (Hero).
- `login.html` / `cadastro.html`: Área do usuário.
